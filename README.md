# BTC Cycle Top Radar

A simple Bitcoin cycle monitoring script that estimates the probability of a market top.

## Indicators

- Pi Cycle Top
- Rainbow Model
- MVRV Approximation
- Puell Multiple Approximation

## Output Example

BTC Top Watcher: NORMAL  
Top Probability: 5% (LOW)

Indicator Breakdown
- Pi Cycle: 4%
- Rainbow Top: 5%
- MVRV Approx: 5%
- Puell Approx: 7%
